Utilização de um formulário:

  • Form é a tag de criação do formulário

  • O action faz o caminho para onde os dados do formulário serão enviados quando o formulário for submetido

  • Method: existem duas funções principais, podendo ser "post" ou "get",Get:Os dados do formulário são anexados à URL especificada no atributo action, resultando em uma URL. Post: são enviados no corpo da solicitação HTTP, tornando-os mais seguros e adequados para enviar grandes quantidades de dados ou dados sensíveis assim como as senhas.


  • O label é como se fosse uma div que estivesse rotulando com o nome da função, Na parte que é inserida alguma informação

  • Já o input email cria uma caixa inteligente para o campo de email, logo assim ele puxa o histórico de emails salvos em cada navegador

  • O aria-label coloca uma audiodescrição do campo
